-> JMeter : installation <-
========


<br>


* download du binaire :

```
http://miroir.univ-lorraine.fr/apache//jmeter/binaries/apache-jmeter-5.1.1.zip
```

et unzip :

```
unzip apache-jmeter-5.1.1.zip -d /usr/bin/jmeter
```

attention: JAVA_HOME

```
/usr/lib/jvm/java-11-openjdk-amd64/
```
 
* install apt

```
apt-get install jmeter
```
